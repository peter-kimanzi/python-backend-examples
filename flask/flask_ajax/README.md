# Flask ajax example

## Install
```shell
mkdir flask_ajax
cd flask_ajax
python3 -m venv venv
source ./venv/bin/activate
pip install Flask==2.0.1
```

## Run
```shell
export FLASK_ENV=development
flask run
```
